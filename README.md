## Описание задания
Подключить внешние библиотеки с помощью Composer и интегрировать в свой проект публичный API без регистрации.

---

## Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd lab
Запустить контейнеры:
```bash
docker-compose up -d --build
```
Открыть в браузере:
```http://localhost:8080```

Содержимое проекта:

```docker-compose.yml``` — описание сервиса Nginx

```www/form.html``` — главная HTML-страница c формой регистрации

```www/index.php``` — страница для вывода данных из сессии и данных из API

```www/process.php``` — обработка данных формы, сохранение в сессию и получение данных из API

```www/view.php``` — страница для просмотра всех сохранённых записей из data.txt

```www/data.txt``` — файл для хранения данных

```www/ApiClient.php``` — класс для работы с публичным API (Guzzle)

```www/UserInfo.php``` — класс для получения информации о пользователе

```composer.json``` — файл конфигурации Composer с зависимостями проекта

```composer.lock``` — файл фиксации версий установленных зависимостей

```vendor/``` — папка с установленными библиотеками Composer (включая Guzzle)

```default.conf``` — конфиг c добавленной секцией для обработки PHP
